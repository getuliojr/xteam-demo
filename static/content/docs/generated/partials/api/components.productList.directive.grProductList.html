grProductList
directive in module components.productList

Description

This directive will render a table with data of the products. It is also responsable to control order by of itens and infinite scroll

Usage

as element:
<gr-product-list
       [page-size="{int}"]>
</gr-product-list>

Parameters

ParamTypeDetails
page-size
(optional)
int

How many records to return each time it goes to the backend. Defaults to 15.
