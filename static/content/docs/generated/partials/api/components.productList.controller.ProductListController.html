ProductListController
controller in module components.productList

Description

It has all the logic for the component productList

Dependencies

  • product

Methods

  • init()

    This is a private function that is called when the controller is initialized and everytime the orderBy changes

  • loadMore()

    This is a private function and it is responsable to load a new batch of records. It is not responsible to show in the screen, only load

  • mergeRecords()

    This is a private function that is responsible to check if it is required to merge the new batch of products to the current list

    It also loads a new batch as soon as it merges the data successfully

  • onOrderChange(orderBy)

    This is a public function that tells the component that the order of the list has been changed by the user It restart the list with the new values

    Parameters
    ParamTypeDetails
    orderBystring

    The name of the field to sort the itens

  • showMore()

    This a public function that tells the component to render a new batch of files.
