Coverage Tests for the Front-end

Karma

To run the coverage tests for this solution you need to have KARMA runner installed in your system. It is recommended to install also the karma-cli as global, that way you can run it anywhere. To install:

    npm install -g karma-cli

Now to run the coverage test, type:

grunt karma:unitCoverage

HTML Report

The tests are part of documentation, as so when the docs are generated, the coverage report is created in HTML format. To check this version, follow the link below:
